Terms Definition
cd Command that is used to change directories
pwd Displays current working directory ( where you're currently working )
mkdir Creates a new directory
nano Command that allows you to create a new text file
cp -r Command that allows you to copy recursively ( copies contents of directories )
rm -fr Command that allows you to remove a file or directory forcefully
sudo Temporarily gives super-user access on a command ( ex: sudo git pull )
